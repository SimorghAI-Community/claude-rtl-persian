# Contributing to Claude RTL Persian

Thank you for helping improve Persian language accessibility in AI platforms.

## Ways to contribute

You can contribute by:

- Reporting bugs
- Suggesting new features
- Improving RTL detection logic
- Adding support for new websites
- Improving documentation
- Translating content
- Testing the extension

## How to contribute code

1. Fork this repository
2. Create a new branch:
3. Make your changes
4. Test locally in Chrome
5. Submit a Pull Request

## Before submitting a Pull Request

Please make sure:

- The extension works correctly
- Existing functionality is not broken
- Your changes are clearly explained

## Community

This project is built for Persian-speaking users and welcomes contributions from developers, researchers, and AI enthusiasts.
