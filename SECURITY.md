# Security Policy

## Reporting Security Issues

If you discover a security vulnerability in Claude RTL Persian, please report it responsibly.

Please do not publicly disclose security issues before they are reviewed by maintainers.

## Supported Versions

| Version | Supported |
|---|---|
| Latest release | ✅ |
| Older versions | ❌ |

## Scope

Security reports related to:

- Extension permissions
- Data privacy
- Script injection issues
- Browser security concerns

are welcome.

Thank you for helping keep this project secure.
